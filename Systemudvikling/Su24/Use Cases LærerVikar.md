# Use cases for LærerVikar 

Aktører: 

Bestillingsansvarlig,

Rekruteringsansvarlig, 

Vikarer, 

Folkeskole.

## Use Case 1
**Titel**: Bestil Vikar

**Primær aktør**: Bestillingsansvarling

**Sekundære aktører**: Vikarer, Folkeskole.  

**Hovedscenarie:**

1. Bestillingsansvarlig vælger "Bestil Vikar"
2. Bestillingsansvarlig vælger dato og tidspunkt
3. System viser en oversigt over ledige vikarer 
4. Bestillingsansvarlig vælger vikar og knytter den til en bestilling
5. Systemet gemmer



## Use Case 2 
**Titel**: Tilføj Vikar

**Primær aktør**: Rekruteringsansvarlig

**Hovedscenarie:**

1. Rekruteringsansvarlig vælger "Tilføj ny vikar"
2. Rekruteringsansvarlig tilføjer information om vikar; 
3. System opretter en ny Vikar instans med information 
4. System gemmer

#### information om vikar;

Navn, alder, køn, fag, tidligere erfaring, børneattest. 


## Use Case 3
**Titel**: Indsæt mulige arbejdstimer 

**Primær aktør**: Vikarer 

**Hovedscenarie:**

1. Vikar vælger "arbejdstimer"
2. Systemet viser en månedsoversigt 
3. Vikar vælger hvilke dage de kan arbejde samt timer
4. Systemet gemmer
5. Systemet videresender information til Bestillingsansvarlig

## Use Case 4
**Titel**: Opsæt vagtplan

**Primær aktør**: Bestillingsansvarlig 

**Hovedscenarie:**

1. Bestillingsansvarlig vælger "ny vagtplan" og vælger periode
2. Systemet opretter en ny vagtplan for perioden
3. Bestillingsansvarlig godkender
